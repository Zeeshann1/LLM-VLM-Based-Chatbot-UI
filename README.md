# LLM and VLM Based Chatbot Web-Application

<img width="320" src="assets/1.jpg">

#### Features:
- Chat with documents
- RAG & Embeddings 
- Chat with images
- llava image API
- Chatbot Q&A
- Chat with any website
- Summarize any website

#### LLM & VLM:
- Lama3.2:3B
- Lama3.2-vision:11B
- Mistral:7B
- Qwen2:7B-instruct & Qwen2.5
- llava:13B
- Minicpm-v:8B
- Deepseek-r1 (comming soon)
- nomic-embed-text:latest
